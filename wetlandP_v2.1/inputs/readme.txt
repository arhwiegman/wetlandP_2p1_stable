#Input files
input data files are given the prefix "df."
two kinds of input files are given to the model `hydroclimate` and `parameters`
The following acronyms are used for field sites:
ID | Site Name 
---| ---
LC | Prindle Road (Prindle brook tributary of Lewis Creek)
OCSP | Otter Creek Swamp Road
OCD | Otter Creek Union Street

The hydroclimate files for these sites provided in the main directory of the inputs folder
